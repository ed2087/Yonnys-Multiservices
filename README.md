## Project Overview

One-page professional handyman website designed for Facebook sharing and local Charlotte, NC market. Features modern design, mobile-first responsive layout, parallax effects, and powerful SEO optimization.

## Maintenance

### Update Contact Info
Edit these sections in `index.html`:
- Line ~56: WhatsApp link
- Line ~58: Phone number display
- Line ~286: Bottom CTA WhatsApp link
- Line ~288: Bottom CTA phone number
- Line ~290: Email link

### Update Services
Edit service cards starting at line ~75 in `index.html`

### Update Service Area
Edit area grid starting at line ~236 in `index.html`

### Update Logo
Replace `images/logo.png` with new logo (max 450px width recommended)

### Update Images
Replace service images in `images/` folder:
- Keep same filenames for easy swap
- Recommended size: 800x600px
- Format: JPG (optimized)
- Max file size: 200KB each
